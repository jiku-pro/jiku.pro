About us

We develop, curate, and customize advanced statistical software tools for continuum data analysis.

Segmentation

Automatically extract data-of-interest from background noise.

Registration

Powerfully align complex geometries to common domains.

Hypothesis testing

Conduct formalized continuum-level hypothesis testing.

Machine learning

Discover and exploit hidden patterns in your data.

Graphical Programming

Customize analyses with mouse-based programming.

Robustness Analysis

Quantify sensitivity and calculate robust effect probabilities.

Electromyography

Meet the international standards for surface and fine-wire EMG analysis.

Reference Data

Compare to worldwide databases to accelerate development.

Open Core

Leverage powerful, world-class open-source computational tools.

Cloud Computing

Store data securely, reduce complex analysis CPU time.
